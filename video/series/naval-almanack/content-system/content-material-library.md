# 内容原料库 v1

这份库负责沉淀《纳瓦尔宝典》系列的内容燃料。它不是选题清单，而是选题、脚本和视觉可以反复调用的“高质量原料卡”。

## 使用原则

- 每条视频至少引用 1 张原料卡，不能从概念直接开写。
- 原料卡必须回答：观众焦虑、反直觉判断、案例种子、行动落点。
- 如果只提供金句，没有现实冲突和案例种子，不允许进入 G0。
- 每次发布复盘后，优先把高价值评论、反驳和误解补回原料库。

## 原料卡模板

| 字段 | 要求 |
| --- | --- |
| 原料编号 | `M-原则-序号`，例如 `M-RETIRE-001` |
| 对应原则 | 退休 / 杠杆 / 具体知识 / 判断力 / 欲望 / 幸福 / 长期主义 |
| 原始表达 | 纳瓦尔原文、改写句或用户评论，不超过 120 字 |
| 当下焦虑 | 观众现在具体卡在哪里，必须能落到工作、钱、时间或关系 |
| 默认误判 | 观众通常怎么想 |
| 反直觉判断 | 这条视频要替观众换成什么判断 |
| 案例种子 | 具体人物、场景、动作、代价 |
| 可视化证据 | 日历、账单、聊天、任务板、资产表、选择分叉等 |
| 行动落点 | 今天或 7 天内能做什么 |
| 记忆句候选 | 一句能被转述的话 |
| 不能讲成 | 书摘、鸡汤、课程广告、泛泛建议 |
| 发布钩子假设 | 抖音 / 视频号 / 小红书上靠什么停留 |

## 高优先级原料卡

### M-RETIRE-001 退休不是年龄，是停止被迫工作

- 对应原则：退休
- 原始表达：退休不是某个年龄，而是你不再为了未来牺牲今天。
- 当下焦虑：很多人把自由全部押到 45 岁、50 岁或 60 岁，却每天继续用睡眠、周末和好奇心还债。
- 默认误判：只要钱存够，我就自由了。
- 反直觉判断：真正的自由不是停止工作，而是现在就减少被迫牺牲的部分。
- 案例种子：34 岁运营负责人每月存下 60%，但连续半年周末回消息；她以为问题是钱不够，其实是时间边界一直被透支。
- 可视化证据：日历像债务表，晚间消息堆叠，退休审计清单。
- 行动落点：写下三件让你不想开始这一周的事，选一件删掉、定价、委托或重新谈边界。
- 记忆句候选：真正的退休，不是不工作，是今天不再被明天绑架。
- 不能讲成：提前退休教程、理财课、躺平宣言。
- 发布钩子假设：从“你可能并不是想退休”切入，比“如何早点退休”更有停留。

### M-LEVERAGE-001 努力不稀缺，能放大的努力才稀缺

- 对应原则：杠杆
- 原始表达：财富来自杠杆，劳动力、资本、代码和媒体都可以放大判断。
- 当下焦虑：很多人每天很忙，但忙完以后没有资产沉淀，明天仍要重新卖时间。
- 默认误判：只要更努力，就会更值钱。
- 反直觉判断：没有杠杆的努力，常常只是更高级的体力劳动。
- 案例种子：一个人反复做内部汇报，另一个人把同样经验写成公开模板、工具或课程入口。
- 可视化证据：两列对照，一边工时堆高，一边资产复利增长。
- 行动落点：把本周重复做三次以上的工作，改成模板、清单、自动化或公开作品。
- 记忆句候选：努力要先找到放大器，否则只是把自己用得更快。
- 不能讲成：鼓吹投机、只讲赚钱技巧。
- 发布钩子假设：用“你不是不努力，是努力没有被放大”开头。

### M-SPECIFIC-001 具体知识，是别人无法外包你的理由

- 对应原则：具体知识
- 原始表达：具体知识无法被训练出来，只能通过好奇心、实践和长期积累获得。
- 当下焦虑：AI 和外包让通用技能越来越便宜，很多人担心自己随时可替代。
- 默认误判：我需要再多学几个通用技能。
- 反直觉判断：越通用越容易被替代，越具体越能积累议价权。
- 案例种子：同样会写方案，一个人只会套模板，另一个人懂行业、客户、组织政治和成交节奏。
- 可视化证据：技能货架 vs 个人地图，通用能力价格下滑曲线。
- 行动落点：写下你过去三年反复被别人请教的问题，把它变成个人方法论。
- 记忆句候选：你真正值钱的部分，往往不是证书，而是别人学不会你的路径。
- 不能讲成：反 AI、反学习、天赋论。
- 发布钩子假设：从“AI 先替代的不是笨人，是可复制的人”切入。

### M-JUDGMENT-001 判断力比勤奋更贵

- 对应原则：判断力
- 原始表达：清晰的判断力，是杠杆能放大的真正东西。
- 当下焦虑：很多人把所有问题都当执行力问题，结果越努力越偏。
- 默认误判：我差的是行动力。
- 反直觉判断：如果方向错，执行力越强，损失越大。
- 案例种子：团队连续三个月加班优化一个没人真正需要的功能，只因为一开始没有问清楚用户为什么要用。
- 可视化证据：罗盘与跑步机、路线图重算、错误方向上的速度表。
- 行动落点：本周所有重要任务先写“我为什么确信这件事值得做”。
- 记忆句候选：勤奋负责加速，判断力负责别把你送错地方。
- 不能讲成：否定行动、只讲思考。
- 发布钩子假设：用“你不是太懒，可能是太快开始了”开头。

### M-DESIRE-001 少一点欲望，本身就是自由

- 对应原则：欲望
- 原始表达：欲望是你和自己签下的契约，直到满足之前都不会快乐。
- 当下焦虑：收入提高以后，焦虑并没有下降，因为生活成本和比较对象同步升级。
- 默认误判：我需要赚更多，才会轻松。
- 反直觉判断：有些自由不是靠赚来，而是靠不再被某个欲望牵走。
- 案例种子：升职后换更贵房子、更贵圈层、更贵消费，表面升级，实际退回更高压力。
- 可视化证据：欲望订阅账单、生活成本阶梯、比较对象滑块。
- 行动落点：列出一个最近最耗你的欲望，判断它来自真实需要还是比较。
- 记忆句候选：有些自由，不是多拥有，而是少被牵引。
- 不能讲成：清贫道德、反消费。
- 发布钩子假设：用“你以为你在赚钱，其实是在养一个更贵的自己”开头。

### M-HAPPINESS-001 幸福不是兴奋，是不被牵着走

- 对应原则：幸福
- 原始表达：幸福是一种没有缺口的状态，不是不断追逐下一次刺激。
- 当下焦虑：刷短视频、买东西、完成目标后只能短暂兴奋，很快又空。
- 默认误判：我需要更多刺激和成就。
- 反直觉判断：幸福常常不是增加刺激，而是减少被外界牵走的次数。
- 案例种子：一个人每晚刷到凌晨，第二天更疲惫，于是继续靠刺激撑住。
- 可视化证据：刺激曲线、通知牵引线、注意力账户。
- 行动落点：给一天设置一个 30 分钟无输入窗口，只观察自己想抓什么。
- 记忆句候选：你不被每个刺激拉走时，才开始拥有自己。
- 不能讲成：冥想课程广告、抽象幸福学。
- 发布钩子假设：从“你不是不快乐，你是一直被打断”切入。

## 维护节奏

- 每制作 1 条视频，至少新增或更新 1 张原料卡。
- 每完成 1 次发布复盘，把高价值评论写入对应原料卡的“观众原话”。
- 每月清理一次，把低质量原料标记为暂缓，避免污染选题判断。
