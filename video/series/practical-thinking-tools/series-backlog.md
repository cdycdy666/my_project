# 系列选题池

| 状态 | 主题 | 核心场景 | 备注 |
| --- | --- | --- | --- |
| 待做 | 二阶思维：别只问马上会怎样 | 冲动辞职、买课、换方向 | 适合第一条，观众代入强 |
| 待做 | 机会成本：你选择的不是 A，而是放弃 B | 时间安排、职业选择 | 可做强行动卡 |
| 待做 | 反向思考：先问怎样一定会失败 | 项目启动、学习计划 | 画面可做失败清单 |
| 待做 | 决策日志：别让事后聪明骗自己 | 投资、跳槽、关键决定 | 易形成系列资产 |
| 待做 | 费曼技巧：讲不清就是没真懂 | 学习、汇报、面试 | 平台传播性强 |
| 待做 | 80/20 法则：别把勤奋用在低杠杆处 | 工作优先级 | 和时间审计可联动 |
| 待做 | 事前验尸：提前写失败原因 | 项目复盘、创业、考试 | 很适合视觉化 |
| 待做 | 沉没成本：已经付出的不该替你做决定 | 感情、工作、投资 | 情绪强，但要避免鸡汤 |
| 待做 | 系统优于目标：别只靠热情冲刺 | 健身、写作、学习 | 适合长期训练主题 |
| 待做 | 第一性原理：把问题拆到不能再拆 | 产品、职业、学习 | 概念熟，但要讲具体 |

## 入池标准

- 必须有真实生活或工作场景。
- 必须能压成一句“旧判断 -> 新判断”。
- 必须能给出一个 7 天内可执行动作。
- 不做纯概念解释，不做工具名词堆砌。
