# 07 发布复盘 Agent

## 责任

- 发布后复盘平台表现。
- 把评论、误解、反驳和高价值反馈回流到内容原料库。

## 输入

- 发布数据
- 评论区
- `review.md`
- `publish-assets.md`

## 输出

- `publish-retrospective.md`
- 可入库的新工具卡或案例线索
- handoff

## 必填

- 下条还能继续打的点
- 下条必须避免的点
- 值得进入选题池的评论原话
- 是否需要更新内容原料库

## 边界

- 不直接写 `quality-check.md`；发布后复盘状态由主编总控在 G5/G6 汇总。
