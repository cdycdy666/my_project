# 脚本实验室

《实用思维工具》的脚本实验室用于在正式写稿前，先低成本比较不同讲法。

## 必做实验

- 5 个开头钩子
- 3 个案例角度
- 2 个错误用法或常见误解
- 3 句记忆句
- 1 个本周行动

## 评分

| 维度 | 分值 |
| --- | ---: |
| 前 3 秒是否像真实问题 | 20 |
| 工具一句话是否清楚 | 20 |
| 案例是否具体可代入 | 20 |
| 错误用法是否能防误解 | 15 |
| 行动是否能当天开始 | 15 |
| 记忆句是否可转述 | 10 |

总分低于 90 不进入正式脚本。

## 脚本失败模式

- 像百科定义，不像解决问题。
- 讲了工具名，却没演示怎么用。
- 案例太抽象，没有人物、时间、动作和代价。
- 行动太大，观众看完仍然不知道第一步。
- 记忆句漂亮但不可执行。
