# Verbal Expression Coach MVP

一个从零搭建的表达模仿教练最小可用版本。

## 当前能力

- 上传目标视频和模仿视频
- 用 SQLite 保存练习记录
- 生成结构化分析结果
- 在前端页面查看历史记录和报告

## 本地启动

```bash
python3 -m uvicorn app.main:app --reload
```

打开 [http://127.0.0.1:8000](http://127.0.0.1:8000)。

## 当前限制

- 分析仍是占位逻辑，还没有接入 ASR、音频特征提取和多模态模型
- 视频文件会保存在 `data/uploads/`
- 练习记录会保存在 `data/coach.db`

## 建议的下一步

1. 接入音频抽取和语音转写
2. 做文本分句与句子对齐
3. 补充语速、停顿等客观指标
4. 接入多模态大模型生成正式教练反馈
