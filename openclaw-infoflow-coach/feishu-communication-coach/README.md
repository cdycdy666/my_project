# Feishu Communication Coach Minimal Modules

这个目录保留了晚间如流沟通复盘会用到的最小模块：

- `feishu_communication_coach/llm.py`：OpenAI-compatible 大模型调用。
- `feishu_communication_coach/references.py`：参考资料读取和相关片段检索。
- `reference-notes/communication-principle-cards.md`：沟通原则卡片库。
- `reference-notes/communication-principle-cards.json`：结构化卡片备份。

完整书籍、私人摘录和运行产生的记录不要提交到 Git。
