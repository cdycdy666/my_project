# Reference Library

把沟通表达相关书籍的摘录、章节笔记或你自己的总结放在这个目录。

支持格式：

- `.md`
- `.txt`
- `.epub`

建议每本书一个文件，例如：

```text
references/nonviolent-communication-notes.md
references/crucial-conversations-excerpts.txt
references/nonviolent-communication.epub
```

文件内容建议使用这种结构：

```markdown
# 书名

## 原则：先观察，再表达感受和请求

摘录或你自己的总结...

## 适用场景：冲突沟通

...
```

机器人会根据你本轮练习内容检索相关片段，并把片段作为教练参考。请只放你有权使用的内容；如果是完整书稿或私人资料，本目录默认被 `.gitignore` 忽略。
